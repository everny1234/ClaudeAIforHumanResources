Questions

Everything you need to know

Do I need a Claude subscription to take this course?
You'll need access to Claude — the free tier works for most exercises, but Claude Pro ($20/month) is recommended for the full experience, especially for longer sessions and file analysis.
I'm not very technical — is this course for me?
Absolutely. This course requires zero coding or data science background. If you can write an email, you can learn to prompt Claude effectively. Module 1 starts from scratch.
How long will it take to complete?
Each module is designed for 45-minute focused sessions. You can complete the full course in one weekend or spread it over 6 weeks — one module per week.
Will this course be updated as Claude evolves?
Yes. Enrolled students get all future updates at no additional cost. AI moves fast and the course will try to keep pace.
